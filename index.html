sega2 css framework

О sega2

Sega2 это небольшая css-библиотека на основе Font Awesome, и шрифта PT Sans. Базовый элемент — компонент, верстается тегом или классом, стили задаются в components.less. Для сброса оформления по-умолчанию используется sanitize.css.

Вариации элемента задаются модификаторами (класс с префиксом «-»). Базовые модификаторы:

  • Размера: -small, -mid, -big, -extra
  • Состояния: -disabled, -active, -selected, -open, -collapsed, -hide, -visible
  • Положения: -sticky
  • Перекрытия: -ellipsis, -overflow-hidden
  • Приоритета: -misc
  • Ширина: narrow, mid, wide, full
При создании кастомного модификатора, например, для -sticky используется след. порядок: -sticky-header.

Библиотека построена на less-файлах:

  • less/colors.less — цветовая гамма
  • less/grid.less — сетка
  • less/typographic.less — заголовки и прочее
  • less/components.less — таблицы, формы, хидер, футер и т. д.
  • less/material.less — тени и некоторые базовые элементы из material design

Установка

bower install sega2 Подключите на своей странице sega2.css <link rel="stylesheet" href="bower_components/sega2/sega2.css" /> или импортируйте себе файл sega2.less, скорее всего придётся в файле sega2.less поправить пути импорта css-файлов. @import "bower_components/sega2/less/sega2.less";
